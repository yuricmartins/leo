# Leo 🦁

A tiny macOS menu bar app that tracks your baby's progress.
```
🦁 Week 17  158 days left
```

No Xcode project. No dependencies. One Swift file.

## Install
```bash
git clone https://github.com/yuricmartins/leo.git
cd leo
chmod +x build.sh && ./build.sh
```

**Requirements:** macOS 13+ and Xcode Command Line Tools (`xcode-select --install`).

## Uninstall
```bash
killall Leo 2>/dev/null
rm -rf /Applications/Leo.app
```
